TypeScript Grundlagen

Vom JavaScript zum typsicheren Code

Über diesen Kurs

In diesem Kurs lernen Sie die grundlegenden Sprachkonzepte von TypeScript kennen. Die Inhalte werden anhand eines durchgängigen Beispielprojekts erläutert und in einem eigenständigen Übungsprojekt praktisch angewendet.

Beispiele

Beispielprojekt: GeoShop

Die Beispiele begleiten den fiktiven Outdoor-Onlineshop GeoShop. Anhand von Produkten, Preisen, Bestellungen und Lagerbeständen lernen Sie die neuen TypeScript-Konzepte zunächst in fertigen und nachvollziehbaren Programmen kennen.

Übungen

Übungsprojekt: Stadtbibliothek Lindenfels

In den Übungen entwickeln Sie schrittweise eine kleine Verwaltungssoftware für die Stadtbibliothek Lindenfels. Sie verwalten Bücher und Mitglieder, erfassen Ausleihen und stellen dafür geeignete Typen, Funktionen und Klassen bereit. Die Aufgaben bauen fachlich aufeinander auf, enthalten jedoch jeweils den vorbereiteten Code, den Sie für die Bearbeitung benötigen.
